Introdução

Bem-vindo à documentação do Mini Framework PHP. Este framework é modular, simples e escalável.

Instalação

Para instalar, execute os seguintes comandos:

git clone https://github.com/seu-repositorio/mini-framework.git
cd mini-framework
composer install

Roteamento

Exemplo de configuração de rotas:

return function (FastRoute\RouteCollector $router) {
    $router->addRoute('GET', '/', 'HomeController@index');
};

Controllers

Exemplo de um Controller:

namespace App\Controllers;

class HomeController {
    public function index() {
        return view('home');
    }
}

Serviços

Os serviços são registrados e gerenciados através do container de injeção de dependências.

namespace App\Services;

class ExemploService {
    public function executar() {
        return "Serviço funcionando!";
    }
}

Kernel

O Kernel gerencia o ciclo de vida da aplicação.

namespace App;

class Kernel {
    public function handle() {
        // Lógica do Kernel
    }
}

Providers

Os providers registram serviços na aplicação.

namespace App\Providers;
use Illuminate\Support\ServiceProvider;

class AppServiceProvider extends ServiceProvider {
    public function register() {
        $this->app->singleton('servico', function() {
            return new \App\Services\ExemploService;
        });
    }
}

Exceptions

Gerenciamento de exceções personalizadas.

namespace App\Exceptions;
use Exception;

        // Lógica de relatório de erro
    }

Console

Criação de comandos personalizados.

namespace App\Console;
use Illuminate\Console\Command;

class ExemploCommand extends Command {
    protected $signature = 'exemplo:executar';
    protected $description = 'Executa um comando de exemplo';

    public function handle() {
        $this->info("Comando executado!");
    }
}
