Instalação

Requisitos:

  • PHP 8+

  • Composer

  • MySQL ou PostgreSQL

Passos:

  1. Clone o repositório:

    git clone https://github.com/seu-repo/framework.git
cd framework

  2. Instale as dependências:

    composer install

  3. Configure o arquivo .env:

    APP_ENV=local
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_DATABASE=framework
DB_USERNAME=root
DB_PASSWORD=

  4. Execute a aplicação:

    php -S localhost:8000 -t public
